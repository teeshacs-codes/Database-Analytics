# NorthStar Operational Analytics

This repository contains SQL, R, Python, Pandas, NumPy, and MongoDB analytical notebooks developed for the NorthStar Urban Mobility and Logistics case study.

The project investigates operational inefficiencies, customer complaints, delivery performance, route optimisation, hub performance, and scalable NoSQL database design using MongoDB Atlas.

Technologies used:
- SQL in R
- Pandas
- NumPy
- Matplotlib
- Seaborn
- MongoDB Atlas
- PyMongo
- Google Colab
